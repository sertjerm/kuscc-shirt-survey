# 🎽 KUSCC Shirt Distribution System

ระบบแจกเสื้อสหกรณ์ออมทรัพย์มหาวิทยาลัยเกษตรศาสตร์

## 🚀 การเริ่มต้น

### รันโปรเจ็กต์
```bash
npm run dev
```

### เปิดเบราว์เซอร์
```
http://localhost:3000
```

## 🔑 ข้อมูลทดสอบ

### สำหรับสมาชิก
- เลขสมาชิก: `123456`
- เบอร์โทร: `0812345678`
- บัตรประชาชน 3 ตัวท้าย: `123`

### สำหรับเจ้าหน้าที่
- เปิดโหมดเจ้าหน้าที่
- เลขสมาชิก: `999999`
- เบอร์โทร: `0000000000`
- บัตรประชาชน 3 ตัวท้าย: `999`

## 📁 โครงสร้างโปรเจ็กต์

```
kuscc-shirt/
├── src/
│   ├── components/Login/    # หน้าเข้าสู่ระบบ
│   ├── pages/              # หน้าหลักต่างๆ
│   ├── services/           # API functions
│   ├── utils/              # Constants และ helpers
│   └── styles/             # CSS files
├── package.json
└── README.md
```
